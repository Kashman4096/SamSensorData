# Dataset description
## Identifiers
- "activity": The type of activity 
- "subject": The subject ID

## Activity column has 6 types as listed below.
1. WALKING
2. WALKING_UPSTAIRS
3. WALKING_DOWNSTAIRS
4. SITTING
5. STANDING
6. LAYING
